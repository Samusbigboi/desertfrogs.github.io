Backup Script

This Bash script automates daily backups to an external drive.


#!/bin/bash
SRC="/home/user/data"
DEST="/mnt/backup"
rsync -av --delete "$SRC" "$DEST"

Python Web Scraper

A simple Python script to scrape headlines from a news site.


import requests
from bs4 import BeautifulSoup

url = "https://example.com"
html = requests.get(url).text
soup = BeautifulSoup(html, "html.parser")
for h in soup.find_all("h2"):
    print(h.text)
